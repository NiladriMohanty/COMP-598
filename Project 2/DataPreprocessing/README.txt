Instructions

1. Open the python script "tags.py".
2. Save the input csv file in same directory.
3. Enter same csv file name in "tags.py" in place of input file name as 'inputfilename.csv'
4. Enter some file name in place of output file name as 'outputfilename.csv'
5. Script will generate the given part of speech tags from input file.
6. Output file of "tags.py" passed to "processAbstract_WI.m" for removing stop words.
7. "processAbstract_WI.m" file calls "porterStemmer.m" file to lemmatize the abstracts
